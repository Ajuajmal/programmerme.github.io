What is Hacktoberfest?
Hacktoberfest is a month-long celebration of open source software run by DigitalOcean in partnership with GitHub and Twilio.

  • Hacktoberfest is open to everyone in our global community!
  • Five quality pull requests must be submitted to public GitHub repositories.

2 day online and offline Hacktober fest along with git workshops

Basic Termux command line commands (Android mobile phone)

Install Packages
pkg install package_name ;
  • pkg install nano -y
    • This command installs nano on your mobile . Nano is an editor tool that is required to edit files from inside termux terminal.   "-y" will return yes to any prompts.
    Usage : nano filename.txt
    Replace filename.txt with the file you want to edit (sample.txt,program.c etc)
  • pkg install git -y
    • This command installs Git on your mobile . You can check on our blog on how to configure git for details on first steps to do after installing git.
  • These are the basic packages you need to install in order to participate on Hacktober fest.There are a lot of packages that you can install on your phone . You can choose that list and type the command and it will be installed . No stress !!
Create Folder

mkdir folder_name
This command is used to create folder using the command line.Just type mkdir and a space followed by the folder name and the folder will be created.

Create File

touch file_name.extension
This command is used to create files using the command line.Just type touch and a space followed by the file name and the prefered extension of the file(eg:Sample.c,program.cpp,text.txt) and the file will be created will be created. It can later be edited using nano or any other text editor of your choice.

View all the files / folder in the directory

ls
ls command is used to list all the files and folder within the directory which we are currently on. It will list all files and folders excluding the hidden ones which can also be shown using the -a parameter .
ls -a

View Path of working directory

pwd
The pwd command can be used to see the path of thw working directory.The usage is very simple just type pwd and press enter.
Done !!

Change directory

cd folder_name
This command is used to change directory (or simply move into a folder) .Just type cd and space and then the folder name . This will change the current directory to the folder that you have typed in . To come out of a folder we can use the command :
  cd ..
The command is cd followed and then a space and then ".."
This command acts like a back button of sorts , the cd folder_name command acts as double-click

Rename Folder

mv old_folder_name new_folder_name
This command is used to rename folder using the command line.Just type mv and a space followed by the name of the you want to rename and then another space and the folder name you want to rename it to.

Delete Folder

rmdir folder_name
This command is used to delete folder using the command line.Just type rmdir and a space followed by the folder name and the folder will be deleted.
