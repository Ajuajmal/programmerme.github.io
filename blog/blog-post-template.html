HacktoberFest :NSS Website and Simple Project Contribution

For participating hacktober first you need to make 5 pull request on or before 31st October 2018, In any of our repo. So in the post I will guide you, for successfull participation

We are going to work on two projects

  • NSS CUCEK- Website (sample: http://nsscucek.ml/) project link https://github.com/ProgrammerMe/nss-cucek
  • Sample Program In Any Language
    • you can craete a sample program and,then pull it our project repo https://github.com/ProgrammerMe/my-first-lang

Steps You Need to make a pull request

  • First goto https://hacktoberfest.digitalocean.com/sign_up/register and register
  • Then Fork One of our repo i.e nss-cucek or my-first-lang using "git clone repourl"
  • cd reponame
  • create a branch using "git checkout -b "branchname"
  • make your changes or create sample program
  • then add the files using "git add file1 file2"
  • commit your changes using "git commit -m "message" "
  • push the changs using "git push -u origin master"
  • goto your github account and then goto the repo you are going to making changes and pull request
  • compare & merge

Each of our projects repo also having clear instructions on "how to make a pull request"

you can make five pull request in same repo. track your status at https://hacktoberfest.digitalocean.com/stats/Yourusername
