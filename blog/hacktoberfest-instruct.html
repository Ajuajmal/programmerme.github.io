Instructions for Participating in Hacktober Fest and Setting up Git and Github

What is Hacktoberfest?
Hacktoberfest is a month-long celebration of open source software run by DigitalOcean in partnership with GitHub and Twilio.

  • Hacktoberfest is open to everyone in our global community!
  • Five quality pull requests must be submitted to public GitHub repositories.

Since it is that time of the year it is time for Hacktober fest. We are going to conduct a 2 day online and offline Hacktober fest along with git workshops.
5 good quality pull request will win a t-shirt .

Instructions For Windows Git Bash Installation

  • Click on the link and Download Git Bash for your 32-bit or 64-bit Operating System
  • Follow Through the normal Installation of windows software by clicking on "next"
  • Open Git Bash
  • Type the following commands

    git config --global user.name "Name"
    Replace Name with the your name and "" are required and press Enter.

    git config --global user.email "Email"
    Replace Email with the your Email and "" are required and press Enter.

Installation Instructions for Android Phones

  • Install Termux from Play Store using this link
  • Open Termux and Long press on Volume up and press q on the keyboard for enabling advanced controls on keyboard.
  • Install nano using the command pkg install nano
  • Install Git using the coomand pkg install git
  • Type the following commands

    git config --global user.name "Name"
    Replace Name with the your name and "" are required and press Enter.

    git config --global user.email "Email"
    Replace Email with the your Email and "" are required and press Enter.

Instructions For Linux Git Installation

  • Open Terminal and type the following commands
    sudo apt-get install git
